# Fix: Preview Panel Text Selection Offset Bug After Scrolling

## Problem

## Root Cause

`g_preview_sel.lines` accumulates ALL lines (scrolled-past + visible), but `g_preview_box` only reflects the visible text lines. In `SelectionRelease`, the code computes `ly1 = anchor_y - box.y_min` which gives an offset within the visible area (0-based), then uses this as an index into `s.lines`. But `s.lines[0]` is the first line of the file (scrolled past), not the first visible line.

Example: If `scroll_y = 5`, `s.lines` has indices 0-4 (scrolled past) + 5-14 (visible). When user clicks at the top of visible area, `ly1 = 0`, but `s.lines[0]` is line 1 of the file, not line 6 (the first visible line).

## Fix

Store `scroll_y` in `TextSelectionState` and use it in `SelectionRelease` to offset the line index.

### File 1: `include/FTB/TextSelection.hpp`

Add `scroll_y` field to `TextSelectionState`:

```cpp
struct TextSelectionState {
    bool active = false;
    int anchor_y = 0, anchor_x = 0;
    int current_y = 0, current_x = 0;
    std::vector<std::string> lines;
    int scroll_y = 0;  // NEW: scroll offset for preview panel
};
```

### File 2: `src/FTB/detail_element.cpp`

Set `scroll_y` on `g_preview_sel` before rendering (around line 558, after `g_preview_sel.lines.clear()`):

```cpp
g_preview_sel.lines.clear();
g_preview_sel.scroll_y = scroll_y;  // NEW: store scroll offset
```

### File 3: `src/FTB/TextSelection.cpp`

In `SelectionRelease()`, adjust line index calculation for preview column (around line 61-64):

```cpp
// Original:
int ly1 = s.anchor_y - box.y_min;
int ly2 = s.current_y - box.y_min;
int line_start = std::max(0, std::min(ly1, ly2));
int line_end   = std::min(static_cast<int>(s.lines.size()) - 1, std::max(ly1, ly2));

// Fixed:
int scroll_offset = (g_active_column == SelectionColumn::Preview) ? s.scroll_y : 0;
int ly1 = s.anchor_y - box.y_min + scroll_offset;
int ly2 = s.current_y - box.y_min + scroll_offset;
int line_start = std::max(0, std::min(ly1, ly2));
int line_end   = std::min(static_cast<int>(s.lines.size()) - 1, std::max(ly1, ly2));
```
