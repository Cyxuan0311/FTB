# FTB 键盘快捷键

## 文件列表导航

| 按键 | 功能 |
|-----|------|
| `j` / `Down` | 向下移动选择 |
| `k` / `Up` | 向上移动选择 |
| `h` / `Left` | 返回父目录（首次 = 第一项，第二次 = 父目录） |
| `l` / `Right` / `Enter` | 进入目录 / 打开文件 |
| `Home` / `g` `g` | 跳转到第一项 |
| `End` / `G` | 跳转到最后一项 |
| `PageUp` | 上一页 |
| `PageDown` | 下一页 |
| `.` | 切换显示隐藏文件 |
| `Space` | 切换批量选择（多选） |

## 文件操作

| 按键 | 功能 |
|-----|------|
| `y` | 复制（yank）选中项 |
| `x` | 剪切选中项 |
| `p` | 从剪贴板粘贴（存在时自动重命名） |
| `P` | 强制覆盖粘贴 |
| `d` | 移动到回收站 |
| `Delete` / `Ctrl+D` | 打开删除确认对话框 |

## 搜索

| 按键 | 功能 |
|-----|------|
| `/` | 进入搜索模式（输入过滤列表） |
| `Escape` | 清除搜索 / 退出搜索模式 |
| `n` | 跳转到下一个匹配 |
| `N` | 跳转到上一个匹配 |
| `Backspace` | 删除最后一个字符 / 搜索框为空时退出 |

## 预览面板滚动

| 按键 | 功能 |
|-----|------|
| `Alt+J` | 预览向下滚动 |
| `Alt+K` | 预览向上滚动 |
| `Alt+H` | 预览向左滚动 |
| `Alt+L` | 预览向右滚动 |
| 鼠标滚轮 | 预览向上/向下滚动 |

## 标签页管理

| 按键 | 功能 |
|-----|------|
| `[` | 上一个标签页 |
| `]` | 下一个标签页 |
| `Ctrl+B nt` | 新建标签页 |
| `Ctrl+B ct` | 关闭当前标签页 |

## 命令模式

按 `Ctrl+B` 进入命令前缀模式，然后输入命令：

| 命令 | 别名 | 功能 |
|---------|---------|------|
| `open` | `op` | 选择程序打开 |
| `openwith` | `ow` | 手动指定程序 |
| `opencfg` | `oc` | 配置打开器 |
| `vim` | `v`、`e`、`editor` | 打开内置编辑器 |
| `theme` | `th` | 打开主题选择器 |
| `rename` | `rn` | 重命名选中项 |
| `batchrename` | `br` | 批量正则重命名 |
| `newfile` | `nf` | 新建文件 |
| `newfolder` | `nd` | 新建文件夹 |
| `preview` | `p` | 完整文件预览 |
| `details` | `d` | 文件夹详情 |
| `jump` | `j` | 跳转到目录 |
| `fdfind` | `fd` | 模糊查找文件 |
| `search` | `s` | 进入搜索模式 |
| `help` | `h` | 帮助面板 |
| `calendar` | `cal` | 日历面板 |
| `layout` | `lo` | 布局设置 |
| `sort` | `so` | 排序模式设置 |
| `uistyle` | `ui` | UI 样式设置 |
| `statusstyle` | `ss` | 状态栏样式 |
| `plugin` | `pl` | 插件管理器 |
| `tasks` | `task`、`ts` | 任务管理器 |
| `clipboard` | `cb` | 查看剪贴板详情 |
| `clr` | `cc` | 清空剪贴板 |
| `newtab` | `nt` | 新建标签页 |
| `closetab` | `ct` | 关闭当前标签页 |
| `nexttab` | `nx` | 下一个标签页 |
| `prevtab` | `pv` | 上一个标签页 |
| `mdsource` | `mds` | 切换 Markdown 预览源码 |
| `xlsx` | `xls` | 切换电子表格预览 |
| `media` | `video`、`mp4`、`gif` | 切换媒体预览 |
| `timg` | `play` | 全屏播放媒体 |
| `pdf` | `hygg` | 切换 PDF 预览 |
| `doc` | `docx`、`pandoc` | 切换 DOC/DOCX 预览 |
| `aud` | `audio`、`eyed3` | 切换音频预览 |
| `hex` | `xxd` | 切换十六进制转储预览 |
| `gh` | | 前往 home 目录 |
| `gd` | | 前往下载目录 |
| `gc` | | 前往配置目录 |
| `ssh` | | SSH 连接（需启用） |

- `Tab` — 自动补全命令名称
- `Enter` — 执行命令
- `Escape` — 取消命令模式

## 内置编辑器（nano 风格）

按 `Ctrl+B v` 或 `Ctrl+B e` 在内置编辑器中打开当前文件。

### 导航

| 按键 | 功能 |
|-----|------|
| 方向键 | 移动光标 |
| `Ctrl+A` / `Ctrl+E` | 行首 / 行尾 |
| `Ctrl+B` / `Ctrl+F` | 左移 / 右移一个字符 |
| `Ctrl+P` / `Ctrl+N` | 上移 / 下移一行 |
| `Ctrl+V` / `PageDown` | 下一页 |
| `Ctrl+Y` / `PageUp` | 上一页 |

### 编辑

| 按键 | 功能 |
|-----|------|
| `Ctrl+O` | 保存文件 |
| `Ctrl+X` | 退出编辑器 |
| `Ctrl+K` | 剪切当前行 |
| `Ctrl+U` | 在光标处粘贴 |
| `Ctrl+W` | 在文件中搜索 |
| `Ctrl+Z` / `Alt+U` | 撤销 |
| `Alt+E` | 重做 |
| `Alt+M` | 切换 Markdown 预览模式 |
| `Ctrl+_` | 跳转到指定行 |
| `Ctrl+C` | 显示光标位置 |

## 全局 / 应用控制

| 按键 | 功能 |
|-----|------|
| `Escape` | 关闭活动面板 / 清除搜索 / 清除批量选择 |
| `Ctrl+C` | 退出 FTB |
| `Ctrl+R` | 重新加载配置文件 |
| `Ctrl+L` | 刷新终端屏幕 |
| `Ctrl+Z` | （忽略，防止意外挂起） |

## 面板（帮助、主题、日历等）

| 按键 | 功能 |
|-----|------|
| `Escape` / `q` | 关闭面板 |
| `Tab` | 切换标签页（帮助面板） |
| 方向键 | 在面板内导航 |
| `Home` / `End` | 跳转到顶部 / 底部 |
| `PageUp` / `PageDown` | 翻页 |

## 鼠标支持

| 操作 | 行为 |
|--------|-----------|
| 左键点击标签栏 | 切换到点击的标签页 |
| 左键点击列 | 开始文本选择 |
| 鼠标拖拽 | 扩展选择 |
| 释放鼠标 | 将选中文本复制到系统剪贴板 |
| 滚轮上 | 预览面板向上滚动 |
| 滚轮下 | 预览面板向下滚动 |
